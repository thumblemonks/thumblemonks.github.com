Riot - Protest the slow test

An extremely fast-running, expressive, and context-driven unit testing framework

Gus wrote Riot as a replacement for Shoulda, Test::Unit, and RSpec; a few of the more popular ruby, unit testing frameworks. Why? Because he wanted a testing framework that was faster than the others, more expressive than the others, and was tailored to fit his slightly different testing style. Riot has been given support for Rack-based app, Mongoid, and ActiveRecord testing. Riot has even been ported to JavaScript.

Smurf

Thoughtless asset minification for Ruby on Rails

Smurf is a Rails plugin that does Javascript and CSS minification the way you would expect. Basically, you hook it up and it does the work. The impetus for this project revolves around laziness; a term Gus likes to refer to as automation. Instead of doing extra work to make file CSS and JavaScript fils sizes smaller, Gus wanted the Rails application to do the work for him.
